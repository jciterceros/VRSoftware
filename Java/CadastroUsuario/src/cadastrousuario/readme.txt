Sistema de Cadastro de Usuario

Campos do cadastro: 
    - Nome da pessoa, idade, sexo, cidade, altura.

Exibir Cadastro:

Opção para mostrar o menu.
    - Realizar Cadastro
    - Exibir Cadastro

Detalhes Técnicos:
    - Implementação utilizando matriz pura.
    - Utilização de funções e métodos estáticos.
    - Utilizar constantes.
    - Abordagem sem orientação a objetos.
    - Inclusão de documentação.

Observação:
    - Não será permitido utilização de ArrayList.


Features Novas:

Adicionar ao Cadastro de Pessoa, um cadastro de Endereço que tera os seguintes atributos:
    - Rua
    - Numero
    - Bairro
    - CEP
    - Cidade
    - Estado
    - Pais
